## What i used
- **React (Next.js)**
- **TS**
- **SASS(SCSS)**
- **Redux**
- **Redux-persist**
- **React tostify**
- **BFF pattern**
## What is it and how to use it
- MPA
- Book shop
- Log in
- Edit profile
- Request books from Google books
- Sort books by categories
- Add books to your favorites
- Manage books in the cart
- Adapted for mobile devices
